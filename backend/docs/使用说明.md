# 使用说明

## 1. 系统概述

本系统是一个基于Go语言开发的后台管理系统，采用前后端分离的架构设计。系统提供了完整的用户管理、角色管理、权限管理、菜单管理等功能，是一个企业级的后台管理解决方案。

## 2. 功能模块

### 2.1 用户管理
- **用户注册**: 新用户可以通过注册页面创建账户
- **用户登录**: 已注册用户可以通过登录页面访问系统
- **用户信息管理**: 管理员可以查看、编辑、删除用户信息
- **密码管理**: 用户可以修改自己的密码
- **状态管理**: 管理员可以启用或禁用用户账户

### 2.2 角色管理
- **角色创建**: 创建新的角色，如管理员、普通用户、编辑等
- **角色编辑**: 修改现有角色的信息
- **角色删除**: 删除不需要的角色
- **权限分配**: 为角色分配相应的权限

### 2.3 菜单管理
- **菜单创建**: 创建新的菜单项
- **菜单编辑**: 修改菜单项的名称、路径、图标等信息
- **菜单删除**: 删除不需要的菜单项
- **菜单层级**: 支持多级菜单结构
- **权限控制**: 根据用户角色显示相应的菜单

### 2.4 操作日志
- **日志查看**: 查看系统中的操作日志
- **日志搜索**: 根据用户、操作内容、时间等条件搜索日志
- **日志删除**: 删除不需要的操作日志

## 3. 操作指南

### 3.1 登录系统
1. 打开浏览器，访问系统地址
2. 在登录页面输入用户名和密码
3. 点击登录按钮
4. 登录成功后会跳转到首页

### 3.2 首页功能
- **数据概览**: 显示系统关键数据统计
- **快速操作**: 提供常用功能的快捷入口
- **导航菜单**: 左侧导航菜单，可以访问各个功能模块

### 3.3 用户管理操作
#### 查看用户列表
1. 点击左侧导航菜单中的"用户管理"
2. 系统会显示用户列表
3. 可以通过搜索框搜索特定用户

#### 添加用户
1. 在用户管理页面点击"新增用户"按钮
2. 在弹出的表单中填写用户信息
3. 选择用户角色
4. 点击确定按钮完成添加

#### 编辑用户
1. 在用户列表中找到需要编辑的用户
2. 点击该行的"编辑"按钮
3. 在弹出的表单中修改用户信息
4. 点击确定按钮保存更改

#### 删除用户
1. 在用户列表中找到需要删除的用户
2. 点击该行的"删除"按钮
3. 确认删除操作

#### 修改用户状态
1. 在用户列表中找到需要修改状态的用户
2. 点击该行的"禁用"或"启用"按钮
3. 确认操作

### 3.4 角色管理操作
#### 查看角色列表
1. 点击左侧导航菜单中的"角色管理"
2. 系统会显示角色列表

#### 添加角色
1. 在角色管理页面点击"新增角色"按钮
2. 在弹出的表单中填写角色信息
3. 点击确定按钮完成添加

#### 编辑角色
1. 在角色列表中找到需要编辑的角色
2. 点击该行的"编辑"按钮
3. 在弹出的表单中修改角色信息
4. 点击确定按钮保存更改

### 3.5 菜单管理操作
#### 查看菜单结构
1. 点击左侧导航菜单中的"菜单管理"
2. 系统会以树形结构显示所有菜单

#### 添加菜单
1. 在菜单管理页面点击"新增根菜单"或某个菜单的"新增"按钮
2. 在弹出的表单中填写菜单信息
3. 选择父级菜单（可选）
4. 点击确定按钮完成添加

#### 编辑菜单
1. 在菜单列表中找到需要编辑的菜单
2. 点击该行的"编辑"按钮
3. 在弹出的表单中修改菜单信息
4. 点击确定按钮保存更改

### 3.6 个人中心
#### 查看个人信息
1. 点击右上角的用户头像或用户名
2. 在下拉菜单中点击"个人资料"

#### 修改密码
1. 点击右上角的用户头像或用户名
2. 在下拉菜单中点击"修改密码"
3. 输入原密码和新密码
4. 点击确定按钮完成修改

#### 退出登录
1. 点击右上角的用户头像或用户名
2. 在下拉菜单中点击"退出登录"
3. 确认退出操作

## 4. 系统配置

### 4.1 后端配置
系统后端配置文件位于 `backend/config/config.yaml`，主要配置项包括：

- **服务器配置**: 端口、运行模式、超时时间等
- **数据库配置**: MySQL连接信息
- **Redis配置**: Redis连接信息
- **JWT配置**: 认证令牌相关设置

### 4.2 前端配置
前端代理配置位于 `frontend/vite.config.ts`，主要配置代理规则将API请求转发到后端服务器。

## 5. 常见问题

### 5.1 登录问题
**问题**: 无法登录系统
**解决方案**:
- 检查用户名和密码是否正确
- 确认后端服务是否正常运行
- 检查网络连接是否正常

### 5.2 权限问题
**问题**: 某些功能无法访问
**解决方案**:
- 确认当前用户是否具有相应权限
- 联系管理员分配相应角色

### 5.3 界面问题
**问题**: 页面显示异常
**解决方案**:
- 清除浏览器缓存
- 尝试使用其他浏览器
- 检查浏览器版本是否兼容

### 5.4 数据问题
**问题**: 数据无法保存或显示
**解决方案**:
- 检查网络连接
- 确认后端服务正常运行
- 联系系统管理员
